# calTrack

A personal health and fitness tracking app built with Next.js. Track your meals, monitor your weight, and plan your workouts — all in a clean dark-themed interface with zero account setup required.

## Features

- **Dashboard** — See today's macro progress at a glance with circular progress rings, current weight, and scheduled workouts
- **Food Log** — Log meals by type (breakfast, lunch, dinner, snack) with full macro tracking (calories, protein, carbs, fat) and date navigation
- **Weight Tracker** — Record daily weigh-ins with a visual progress chart and stats (current, change, total change)
- **Workout Planner** — Create workout routines with exercises, sets, reps, and weights. Includes a barbell plate calculator that shows you exactly how to load the bar

## Tech Stack

- [Next.js](https://nextjs.org) 16 with App Router
- [React](https://react.dev) 19
- [TypeScript](https://www.typescriptlang.org) 5
- [Tailwind CSS](https://tailwindcss.com) 4
- [Recharts](https://recharts.org) for data visualization
- [Lucide](https://lucide.dev) for icons
- localStorage for data persistence (no backend needed)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy

Deploy to [Vercel](https://vercel.com) with one click — no environment variables or backend configuration needed.
